# Расширение для оценки фильмов

Современное расширение для Chrome, позволяющее оценивать фильмы, просматривать рейтинги сообщества и искать фильмы через API Кинопоиска.

## Возможности

- 🎬 **Поиск фильмов** - поиск через API Кинопоиска с фильтрами
- ⭐ **Система оценок** - оценка от 1 до 10 с комментариями
- ✏️ **Редактирование оценок** - изменение и удаление своих отзывов
- 👥 **Лента сообщества** - рейтинги всех пользователей
- 💬 **Отзывы пользователей** - просмотр всех отзывов на странице фильма
- 👤 **Редактирование профиля** - изменение имени, аватара, пароля
- 🔐 **Авторизация** - через Google или Email/Пароль
- 🎨 **Светлая и темная темы** - выбор темы оформления по вкусу
- 📊 **Rated** - управление своими оценками с фильтрами
- 🔖 **Закладки (Bookmarks)** - единый раздел для управления списками (Смотрю, В планах, Избранное) и кастомными коллекциями
- 📻 **Аниме Радио** - встроенный плеер в навбаре с выбором станций (Anison.FM, Radio Nami)
- ⏭️ **Пропуск заставок (AniSkip)** - автоматический и ручной пропуск оппенингов и эндингов в аниме
- ⚙️ **Гибкие настройки** - выбор языка интерфейса, режима отображения (Popup/Side Panel) и других опций
- 🛡️ **Админ панель** - продвинутая дашборд-панель для управления пользователями и контентом

## Как установить расширение в Chrome

### Простая установка (для пользователей)

1. **Скачайте расширение**
   - Разархивируйте скачанный файл проекта
   - Переместите все файлы в одну папку

2. **Откройте Chrome**
   - Введите в адресной строке: `chrome://extensions/`
   - Нажмите Enter

3. **Включите режим разработчика**
   - В правом верхнем углу найдите переключатель "Режим разработчика"
   - Включите его

4. **Загрузите расширение**
   - Нажмите кнопку "Загрузить распакованное расширение"
   - Выберите папку с файлами расширения
   - Нажмите "Выбрать папку"

5. **Готово!**
   - Расширение появится в списке
   - Иконка расширения появится справа от адресной строки
   - Нажмите на иконку, чтобы открыть расширение

### Требования

- Google Chrome версии 88 или выше
- Подключение к интернету

## Как пользоваться

1. **Войдите в систему**
   - Нажмите на иконку расширения
   - Выберите вход через Google или Email
   - Авторизуйтесь

2. **Ищите фильмы**
   - Используйте поиск в popup для быстрого поиска
   - Нажмите "Advanced Search" для расширенного поиска
   - Применяйте фильтры по году, жанру, стране

3. **Оценивайте фильмы**
   - Нажмите на карточку фильма
   - Нажмите "Rate This Movie"
   - Выберите оценку от 1 до 10
   - Добавьте комментарий (необязательно)
   - Нажмите "Save Rating"

4. **Редактируйте оценки**
   - В "Rated" нажмите "Edit Rating" на карточке фильма
   - Или используйте меню (⋮) на своих отзывах
   - Измените оценку или комментарий
   - Нажмите "Save Rating"

5. **Смотрите рейтинги**
   - В главном окне видна лента всех оценок
   - На странице фильма - секция "Оценки пользователей"
   - Нажмите "Refresh" для обновления

6. **Управляйте профилем**
   - Перейдите на страницу своего профиля
   - Нажмите на три точки (⋮) в правом верхнем углу баннера
   - Выберите "Edit Profile" из выпадающего меню
   - Измените имя, аватар, баннер или пароль
   - Сохраните изменения

7. **Управляйте закладками и коллекциями**
   - Откройте раздел "Bookmarks" для доступа ко всем спискам
   - Просматривайте фильмы "Смотрю", "В планах" и "Избранное"
   - Создавайте собственные коллекции с уникальными иконками
   - Добавляйте фильмы в любые списки прямо со страницы фильма

8. **Настройте расширение под себя**
   - Перейдите в раздел "Settings" (Настройки)
   - Выберите язык интерфейса и внешний вид (Popup или боковая панель)
   - Включите "Аниме Радио" для фонового прослушивания музыки

9. **Смотрите с комфортом**
   - Используйте встроенный видеоплеер с поддержкой HLS
   - Для аниме воспользуйтесь функцией пропуска заставок (AniSkip)
   - Плеер сам сохраняет историю просмотра и выбранные субтитры

10. **Админ панель** (только для администраторов)
    - Доступ к расширенным функциям управления
    - Управление пользователями, отзывами и жалобами
    - Модерация контента через удобный дашборд

## Технологии

- Chrome Extension Manifest V3
- Firebase (Authentication + Firestore)
- Kinopoisk API
- Vanilla JavaScript (без фреймворков)
